# STOIC DAILY by Basti

A minimalist daily reflection app built with Python and customtkinter.
Every day you get a unique Stoic question to reflect on. You can also
explore random questions from the pool of 255 reflections.

## Run it
```bash
pip install -r requirements.txt
python main.py
```

## Built with
- Python 3.8+
- customtkinter
- datetime, random (standard library)
